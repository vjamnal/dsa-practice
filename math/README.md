# Math

Mathematical problems and number theory.

## Problems

1. **Reverse Integer** - Reverse digits of an integer
2. **Palindrome Number** - Check if number is palindrome
3. **Power of Two** - Check if number is power of 2
4. **Happy Number** - Determine if number is happy
5. **Plus One** - Add 1 to number represented as array

## Common Techniques

- Modulo and division operations
- Mathematical formulas
- Bit manipulation
- Number properties

## Time Complexity

Usually O(log n) for digit-based problems
