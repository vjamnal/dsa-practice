# Algorithms

This directory contains implementations of common algorithms.

## Available Algorithms

### Sorting
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### Searching
- Linear Search
- Binary Search (iterative and recursive)
- Jump Search

### Dynamic Programming
- Fibonacci
- 0/1 Knapsack
- Longest Common Subsequence
- Coin Change
- Edit Distance

### Recursion
- Factorial
- Power
- Tower of Hanoi
- String operations

## Usage

Import and use any algorithm:

```python
from algorithms.sorting.sorting_algorithms import quick_sort

arr = [64, 34, 25, 12, 22, 11, 90]
sorted_arr = quick_sort(arr)
print(sorted_arr)
```
