# Common Coding Problems

This directory contains solutions to frequently asked coding interview problems.

## Available Problems

### Two Sum
Find two numbers in an array that add up to a target value.
- **Solution 1**: Hash map approach - O(n) time
- **Solution 2**: Two pointers (sorted array) - O(n) time

### Reverse Linked List
Reverse a singly linked list.
- **Solution 1**: Iterative - O(n) time, O(1) space
- **Solution 2**: Recursive - O(n) time, O(n) space

### Valid Parentheses
Check if parentheses in a string are balanced.
- **Solution**: Stack-based approach - O(n) time, O(n) space

### Maximum Subarray
Find the contiguous subarray with the largest sum.
- **Solution**: Kadane's Algorithm - O(n) time, O(1) space

## Running Problems

Each problem file includes test cases and can be run directly:

```bash
python problems/two_sum.py
python problems/valid_parentheses.py
```
