# Data Structures

This directory contains implementations of fundamental data structures.

## Available Data Structures

### Arrays
- **dynamic_array.py**: Dynamic array with automatic resizing

### Linked Lists
- **singly_linked_list.py**: Singly linked list with basic operations

### Stacks
- **stack.py**: LIFO stack implementation

### Queues
- **queue.py**: FIFO queue implementation

### Trees
- **binary_search_tree.py**: BST with traversal methods

### Graphs
- **graph.py**: Graph with BFS and DFS traversal

### Hash Tables
- **hash_table.py**: Hash table with collision handling

## Usage

Import and use any data structure:

```python
from data_structures.stacks.stack import Stack

stack = Stack()
stack.push(1)
stack.push(2)
print(stack.pop())  # 2
```
