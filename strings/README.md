# Strings

String manipulation and pattern matching problems.

## Problems

1. **Valid Palindrome** - Check if string is palindrome
2. **Valid Anagram** - Check if two strings are anagrams
3. **Longest Substring Without Repeating Characters** - Find longest unique substring
4. **Longest Palindromic Substring** - Find longest palindromic substring
5. **Group Anagrams** - Group strings that are anagrams
6. **Reverse String** - Reverse a string in place

## Common Techniques

- Two pointers
- Hash maps for character counting
- Sliding window
- Dynamic programming
