# Arrays

Array-based problems and algorithms.

## Problems

1. **Two Sum** - Find two numbers that add up to target
2. **Best Time to Buy and Sell Stock** - Find maximum profit
3. **Contains Duplicate** - Check if array has duplicates
4. **Product of Array Except Self** - Calculate product array
5. **Maximum Subarray** - Find contiguous subarray with largest sum (Kadane's Algorithm)
6. **Rotate Array** - Rotate array by k steps
7. **Find Minimum in Rotated Sorted Array** - Find minimum element

## Time Complexity

- Access: O(1)
- Search: O(n)
- Insertion: O(n)
- Deletion: O(n)
